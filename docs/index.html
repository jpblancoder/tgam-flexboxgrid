The Globe and Mail's flexbox grid v1.0.14

A mobile-first flexbox grid system based on the CSS flex display property. A customized, namespaced, and extended Bootstrap 4 grid.

It also supports different column counts for each of the 4 breakpoint tiers. Which are extra-small (1), small (8), medium (12), and large (16).

Filesize is 25.11 KB minified, and 3.36 KB gzippped. Github Repo: tgam-flexboxgrid.
